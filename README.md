# ☕ Core Java Programs

A comprehensive collection of Core Java programs covering fundamental to advanced concepts. Organized by topic, this repository serves as a reference and practice ground for Java learners and developers.

---

## 📚 Topics Covered

| Topic | Concepts |
|-------|---------|
| **OOP Concepts** | Classes, Objects, Inheritance, Polymorphism, Abstraction, Encapsulation |
| **Arrays** | 1D & 2D arrays, Array manipulation, Sorting, Searching |
| **Strings** | String methods, StringBuilder, StringBuffer, String manipulation |
| **Collections** | ArrayList, LinkedList, HashMap, HashSet, TreeMap, Iterator |
| **Exception Handling** | try-catch-finally, Custom exceptions, Checked vs Unchecked |
| **Multithreading** | Thread class, Runnable interface, Synchronization, Thread lifecycle |
| **Java Fundamentals** | Data types, Operators, Control flow, Methods, Recursion |

## 🚀 Getting Started

### Prerequisites
- Java JDK 8 or higher
- Any Java IDE (IntelliJ IDEA, Eclipse, VS Code) or terminal

### Run a Program

1. **Clone the repository**
   ```bash
   git clone https://github.com/alekhyaGontla/Core-java.git
   cd Core-java
   ```

2. **Compile and run any file**
   ```bash
   javac FileName.java
   java FileName
   ```
